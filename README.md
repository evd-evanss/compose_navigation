# Navigating between modules with compose

An example of how to implement navigation between modules using Jetpack Compose

## Module dependencies
|- app

    -- :navigation
    
    -- :order
    
    -- :splash
    
    -- :shoppingcart

|- splash

    -- :navigation

|- order

    -- :navigation

|- shoppingcart

    -- :navigation

|- navigation
